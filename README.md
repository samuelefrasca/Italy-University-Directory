# 🎓 Italy University Directory

Un elenco navigabile e intuitivo delle università in Italia. Questo progetto nasce dall'esigenza di avere una mappatura chiara degli istituti statali, sviluppata con un focus sulla semplicità e la velocità di consultazione.

🚀 **[Vai al sito live](https://samuelefrasca.github.io/Italy-University-Directory/)**

## 🛠️ Caratteristiche
*   Visualizzazione completa delle università italiane.
*   Dati organizzati per Regione, Città e numero di studenti.
*   Link diretti ai portali ufficiali di ogni istituto.
*   Interfaccia responsive (ottimizzata per mobile e desktop).

## 💻 Tech Stack
*   **HTML5** & **CSS3** (Layout pulito e moderno).
*   **Vanilla JavaScript** (Manipolazione del DOM e gestione dei dati tramite Array di oggetti).

## 📈 Obiettivi del Progetto
Sono uno studente di Informatica all'Università di Pisa e sto approfondendo lo studio di JavaScript. Questo progetto mi permette di mettere in pratica concetti fondamentali come:
*   Gestione di strutture dati (Array, Objects).
*   Rendering dinamico dei componenti nel browser.
*   Ottimizzazione della qualità dei dati (validazione link e nomi ufficiali).
